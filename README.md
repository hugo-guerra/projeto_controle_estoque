# Sistema de Controle de Estoque de Itens

Sistema para controlar estoques de itens, bem basico para treinar e evoluir as minha abilidade.

## Funcionalidades

- Cadastramento de Item
- Listagem dos Itens
- Buscar
- Atualização de valores
- Remoção
- Ver alerta de estoque baixo

## Como fazer funcionar na sua maquina

1. Clone esse repositório
2. Acesse a pasta desse projeto
3. Execute o comando `python main.py`

## Estrutura do projeto

```
controle_estoque/
|-- main.py
|-- estoque.py
|-- armazenamento.py
|__ dados/
       |__ estoque.json    
```

## Ferramentas Usadas

- Python
- Modulos: os e json
